Роль: Senior Full-stack разработчик и архитектор ИИ-платформ.Проект: ThreadForge — Индустриальный «Контент-завод» для автоматизации Threads.net и Instagram.Язык интерфейса: Русский (обязательно).ОБЩАЯ ЗАДАЧА:Создать SaaS-приложение в стиле «Digital Factory 2026» для массового управления аккаунтами Threads. Главная цель — превращать «сырье» (длинные тексты, новости) в цепочки связанных постов (треды) и публиковать их автоматически через Meta API.1. ВИЗУАЛЬНЫЙ СТИЛЬ (ПРЕМИАЛЬНЫЙ МИНИМАЛИЗМ):Цвета: Фон — абсолютно черный (#000000). Карточки и модальные окна — темно-серый градиент (#0A0A0A → #111111) с ультратонкими границами (0.5px, белые с прозрачностью 15%).Акценты: Неоновый фиолетовый (#8B5CF6) для кнопок ИИ («Парсинг», «Генерация»), ярко-белый для основного текста.Шрифт: Inter. Иконки: Lucide React.2. АРХИТЕКТУРА И ЛОГИКА «ЦЕПОЧЕК» (THREADS CHAINING):Chaining Engine: Реализовать логику связки постов через parent_id. При публикации ветки система сначала постит «Главный пост», получает его ID и использует его как родительский для следующего «Ответа», и так далее до конца цепочки.Динамический конструктор веток: В интерфейсе создания шаблона добавить выпадающий список «Количество веток» (от 1 до 10). При выборе числа $N$ интерфейс должен мгновенно отобразить $N$ текстовых колонок/полей для редактирования каждого звена цепочки.3. МОДУЛИ ИНТЕРФЕЙСА (SIDEBAR & TABS):Sidebar (Навигация): Дашборд, Аккаунты, Шаблоны, AI Генератор, AI Автопостинг, Тест тредов.Главный Дашборд: Сверху StatGrid (4 карточки: «Всего постов», «Опубликовано сегодня», «Запланировано», «Активных аккаунтов») с живыми sparkline-графиками. Ниже — сетка карточек последних шаблонов (поля: Название, Аккаунт, Метка «N веток», Статус «Активен»).Вкладка «Аккаунты»: Таблица управления. Поля: Username, ID профиля, Статус (зеленый/красный индикатор), Таймер жизни токена (60 дней). Форма добавления токена Meta API.AI Генератор (Content Factory): Форма для вставки «сырья» (лонгриды, RSS). Поле «Текст-референс» для анализа стиля. Кнопка «AI Парсинг» (фиолетовая), которая через GPT-4o автоматически распределяет сырье по выбранным веткам, очищая текст и сохраняя смысл.4. МОДАЛЬНЫЕ ОКНА И СИСТЕМНЫЕ ФУНКЦИИ:Окно «Новый шаблон»: Поля «Название», «Выбор аккаунта», кнопка «+ Добавить ответ в тред» и динамический список шагов.Окно «Пакетная публикация»: Выбор интервала между постами в минутах (минимум 5 мин). Авто-расчет: «Общее время работы завода: ~X часов».Переменные: Поддержка тегов {date}, {time}, {day}, {month} для авто-вставки текущих данных в текст.5. ТЕХНИЧЕСКИЙ СТЕК:Frontend: React.js, Tailwind CSS, shadcn/ui.Backend: Node.js.База данных: PostgreSQL (через Prisma) для хранения аккаунтов, токенов, JSON-структур цепочек и логов публикаций.ИИ: Интеграция с OpenAI API для функций трансформации и парсинга контента.Инструкция для Агента: Начни с создания схемы базы данных, поддерживающей иерархию постов (parent_id), и сверстай Sidebar и Dashboard в абсолютно черном премиальном стиле.
